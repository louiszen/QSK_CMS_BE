## <span style="color: #569CD6">static</span> *`Pad`* 
---
```jsx

```

### **Usage**

<br/>

### **Parameters**
| Props | Type | Required | Default | Description |
| :---|:---:|:---:|:---:|:---|
| str | `String` | ✔ || The string to be modified |

| length | `Number` | ✔ || Desirable length |

| :---|:---:|:---:|:---:|:---|
